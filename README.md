Vim as Code Viewer in Sub Monitor
=================================

```vim
" Open current file in subvim.  If there is no subvim, it starts it.
SubvimOpen

" Open some_file in subvim
SubvimOpen path/to/some_file

" Start subvim explicitly.
SubvimStart

" Close subvim
SubvimQuit

" Move subvim window foreground
SubvimForeground
```
